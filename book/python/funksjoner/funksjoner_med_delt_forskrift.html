Funksjoner med delt forskrift#

I matematikken dukker det noen ganger opp funksjoner med delt forskrift. Et eksempel på dette er

\[\begin{split} f(x) = \begin{cases} x^2, & \text{hvis } x \leq 0, \\ x, & \text{hvis } x > 0. \end{cases} \end{split}\]

I Python kan vi definere slike funksjoner ved hjelp av if-else-blokker. For eksempel kan vi definere \(f\) som ovenfor slik:

def f(x):
    if x <= 0:
        return x**2
    else:
        return x

Grafisk kan vi visualere funksjonen for å se at den blir som forventet:

Show code cell source Hide code cell source 
import numpy as np # Importerer numerisk bibliotek
import matplotlib.pyplot as plt # Importerer plottebibliotek

x0 = np.linspace(-2, 0, 1001)
x1 = np.linspace(0, 2, 1001)
plt.plot(x0, [f(i) for i in x0], label="$x^2$")
plt.plot(x1, [f(i) for i in x1], label="$x$")
plt.xlabel("$x$")
plt.ylabel("$f(x)$")
plt.grid(True)
plt.legend()
plt.show()

Vi kan også håndtere mer sammensatte delte forskrifter, som

\[\begin{split} g(x) = \begin{cases} x^2, & \text{hvis } x \leq 0, \\ x, & \text{hvis } 0 < x \leq 1, \\ x^3, & \text{hvis } x > 1. \end{cases} \end{split}\]

som vi kan definere i Python med en if-elif-else-blokk:

def g(x):
    if x <= 0:
        return x**2
    elif 0 < x <= 1:
        return x
    else:
        return x**3

Oppgave 1#

Lag en Pythonfunksjon for funksjonen \(f\) definert under:

\[\begin{split} f(x) = \begin{cases} x^2, & \text{hvis } x \leq 0, \\ -x + 2, & \text{hvis } x > 0. \end{cases} \end{split}\]

Du kan ta utgangspunkt i kodeskallet under. Du må fylle inn der det står NotImplemented.

def f(x):
    if NotImplemented:
        return NotImplemented
    else:
        return NotImplemented
Løsningsforslag

Her er to måter vi kan definere funksjoner på:

def f(x):
    if x <= 0:
        return x**2
    else:
        return -x + 2

eller

def f(x):
    if x > 0:
        return -x + 2
    else:
        return x**2

Oppgave 2#

Lag en Pythonfunksjon for funksjonen \(g\) definert under:

\[\begin{split} g(x) = \begin{cases} x^2 - 2x + 1, & \text{hvis } x \leq -1, \\ x^2 - 3x + 3, & \text{hvis } -1 < x < 0, \\ 2x + 1, & \text{hvis } x \geq 0. \end{cases} \end{split}\]

Du kan ta utgangspunkt i kodeskallet under. Du må fylle inn der det står NotImplemented.

def g(x):
    if NotImplemented:
        return NotImplemented
    elif NotImplemented:
        return NotImplemented
    else:
        return NotImplemented
Løsningsforslag

En mulig løsning her er å definere funksjonen slik:

def g(x):
    if x <= -1:
        return x**2 - 2 * x - 1
    elif -1 < x < 0:
        return x**2 - 3 * x + 3
    else:
        return 2 * x + 1

Oppgave 3#

Lag en Pythonfunksjon for funksjonen \(h\) definert under:

\[\begin{split} h(x) = \begin{cases} 2x + 1, & \text{hvis } x \leq 0, \\ xe^x + 1, & \text{hvis } 0 < x \leq 1, \\ \frac{1}{x}, & \text{hvis } 1 < x \leq 2, \\ \ln(x), & \text{hvis } x > 2. \end{cases} \end{split}\]

Du kan ta utgangspunkt i kodeskallet under. Du må fylle inn der det står NotImplemented.

import math # Importerer matematikk-biblioteket for å kunne bruke exp-funksjonen og log-funksjonen
def h(x):
    if NotImplemented:
        return NotImplemented
    elif NotImplemented:
        return NotImplemented
    elif NotImplemented:
        return NotImplemented
    else:
        return NotImplemented
Løsningsforslag

En mulig løsning er:

def h(x):
    if x <= 0:
        return 2 * x + 1
    elif 0 < x <= 1:
        return x * math.exp(x) + 1
    elif 1 < x <= 2:
        return 1 / x
    else:
        return math.log(x)
