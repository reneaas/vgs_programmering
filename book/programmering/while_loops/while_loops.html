Oppsummering: while-løkker

En while-løkke brukes for å gjenta en kodeblokk så lenge en betingelse er sann. Generelt kan en while-løkke skrives på følgende måte:

while betingelse:
    # Gjøre noe så lenge `betingelse` er sann!

  • betingelse er en logisk test som enten er True (sann) eller False (usann).

Eksempler

Betingelse

Beskrivelse

while x < 5:

x < 5

Gjenta så lenge x er mindre enn 5

while x > 10:

x > 10

Gjenta så lenge x er større enn 10

while x <= 5:

x <= 5

Gjenta så lenge x er mindre eller lik 5

while x >= 8:

x >= 8

Gjenta så lenge x er større eller lik 8

Eksempelkode:

1x = 1
2while x < 5:    # Så lenge x er mindre enn 5
3    print(x)    # Skriv ut verdien til x
4    x = x + 2   # Øk x med 2

Rekursive sammenhenger#

while-løkker egner seg godt for å utforske sammenhenger som er rekursive. En rekursiv sammenheng er ofte gitt ved en formel der det neste tallet i en tallfølge er bestemt av det forrige.
